◆ delete_descendants_from_page()

bool t4::pdf::structure_element::delete_descendants_from_page ( const reference &  page_ref)

Recursively deletes children associated with particular page. For optimization purpose, in case if all children belong to the page (a.k.a. element represents structure of this page only) it does not perfom deletion but returns 'true' indicating that the whole element should be deleted

Parameters
page_refreference to the page that is about to be removed and whose structure elements needs to be deleted
Returns
'true' if element should be deleted from parent, otherwise 'false'
