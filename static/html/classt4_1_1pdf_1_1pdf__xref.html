◆ validate_all_checksums()

xref_checksum_validation_status t4::pdf::pdf_xref::validate_all_checksums ( )

Returns the cumulative validation status for all the "CheckSum" values in the document.

Returns
valid if all the "CheckSum" values in the document match the checksums of their corresponding xref tables, invalid if at least one "CheckSum" value in the document does not match the checksum of its corresponding xref table, not_available if at least one "CheckSum" value in the document is missing, error if no "CheckSum" values could be calculated or processed.

◆ validate_trailing_checksum()

xref_checksum_validation_status t4::pdf::pdf_xref::validate_trailing_checksum ( )

Returns the validation status for only the trailing (last) "CheckSum" value in the document.

Returns
valid if the "Checksum" value matches the checksum of its corresponding xref table, invalid if the "Checksum" value does not match the checksum of its corresponding xref table, not_available if there is no "CheckSum" value, error if the "CheckSum" value could not be calculated or processed.
