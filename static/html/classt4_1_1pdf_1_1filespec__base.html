Detailed Description

Class representation of a PDF file specification, as described in the PDF Reference section 3.10.2 "File Specification Dictionaries."

This is a base class and is typically not used directly by clients. Instead use derived classes filespec and associated_file.

◆ get_filename()

t4::string t4::pdf::filespec_base::get_filename ( ) const

Returns the file name, which is the value of one of the "UF", "F", "DOS", "Mac" or "Unix" keys.

When multiple keys are set the implementation returns the best value, using the following precedence:

  1. "UF"
  2. "F"
  3. "DOS", "Mac" or "Unix" depending on the platform the code is running on
  4. another "DOS", "Mac" or "Unix" key, if set
Returns
The file name.
